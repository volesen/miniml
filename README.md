# miniml

A minimal ML dialect, but with ad-hoc polymorphism through type classes.

- [x] Lexer
- [x] Parser
- [ ] Sweeten syntax
- [x] Basic interpreter
- [x] Recursion
- [ ] Mutual recursion
- [x] Type inference with let-polymorphism
- [ ] Type classes
- [ ] REPL
